Getting this data in the proper format was challenging. Figuring out german translations, as well sifting through different data formats was required to get the data into a valid format. One of the data sets in this case doesn’t have a distinction between female and male data, meaning there may be female heights in the data alongside the male data.

A simple linear regression shows that the heights appear to be decreasing, but a linear model has a really low r squared value, meaning the model is not useful.There is not really any clear analysis to be drawn from the data. I don’t think the data is clean or full enough to make a meaningful conclusion. We would need more data from each country, and that data would have to have more integrity.
